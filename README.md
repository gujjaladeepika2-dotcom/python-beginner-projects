# To-Do List App
## Description
A simple Python To-Do List application that allows you to:
- Add tasks
- View tasks
- Delete tasks

This project is designed for beginners to practice Python basics like lists, loops, and conditionals.
## Features
- User-friendly command line interface
- Stores tasks in memory
- Easy to use and extend
